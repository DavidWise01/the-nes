# THE NES · NES

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **EXEREÚNESIS**.

THE NES (NES) — the first sphere of UD0's new 23rd domain EXEREÚNESIS (ἐξερεύνησις, the searching-out): TECHNICAL TEARDOWNS. A full 8-bit Nintendo (1985) hardware teardown + SPEC EDITION. Almost nothing inside: two custom Ricoh chips, 2 KB RAM, a slot. The 2A03 (MOS 6502 core @ 1.79 MHz + the APU on-die), the 2C02 PPU (256x240, 64 sprites / 8 per scanline = the flicker, 54-color palette ~25 on screen), the 5-channel APU, the 2 KB work RAM, and the CARTRIDGE + MAPPER (MMC1/MMC3 bank-switching that kept upgrading the machine for a decade). Plus the 10NES lockout chip + the wear-prone 72-pin connector (the real roots of the blinking power light), the serial controller, NTSC composite out. ⚠ two-layer: chips/clocks/memory-map/sprite limits are documented FACT; the blinking-light + lockout-arms-race LORE is flagged (well-supported version, not the tall tale). Render-not-invent; NO ROMs/BIOS/code reproduced; Nintendo/Ricoh + NESdev cited not minted. LIVE clickable block-diagram teardown (click a chip -> its spec) + a spec sheet. Vellum (no black) + blueprint-slate + quorum-ring favicon. 12 components.
